# Go Tutorial - Lesson 24: Goroutines

This repository contains the source code for **Go Tutorial for Beginners #24 - Goroutines & Concurrency**.

## Files

- `goroutine_basic.go` - Basic goroutine with the `go` keyword
- `goroutine_multiple.go` - Running multiple goroutines concurrently
- `goroutine_anonymous.go` - Anonymous goroutines and closures
- `goroutine_waitgroup.go` - Using sync.WaitGroup for synchronization

## Running the Examples

```bash
go run goroutine_basic.go
go run goroutine_multiple.go
go run goroutine_anonymous.go
go run goroutine_waitgroup.go
```
